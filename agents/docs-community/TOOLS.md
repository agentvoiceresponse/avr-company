# Documentation & Community Tools

## Available MCP Servers and Tools

### GitHub MCP Server
- Read issues across all repos for triage
- Apply labels and respond to issues
- Manage the avr-docs repository
- Update the .github org profile README

### AVR Docs MCP Server (avr-docs-mcp)
- Search and query the AVR wiki content
- Update wiki pages programmatically
- Check documentation coverage

### Web Search
- Research documentation best practices
- Check competitor documentation quality
- Find community discussions about AVR on forums, Reddit, etc.

### File System
- Create and edit Markdown documentation
- Generate architecture diagrams (Mermaid)
- Manage wiki page templates
