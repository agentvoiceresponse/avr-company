# Backend Developer Tools

## Available MCP Servers and Tools

### GitHub MCP Server
- Clone, branch, commit, and push to all avr-* connector repos
- Create and manage pull requests
- Read and respond to assigned issues

### Code Execution
- Run Node.js and npm commands
- Execute Docker build and test commands
- Run unit tests with npm test

### File System
- Create and edit JavaScript/TypeScript files
- Manage package.json dependencies
- Create Dockerfiles and configuration files

### Web Search
- Read AI provider API documentation
- Research SDK usage patterns and examples
