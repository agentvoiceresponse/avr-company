# DevOps Engineer Tools

## Available MCP Servers and Tools

### GitHub MCP Server
- Manage avr-infra repository (Docker Compose files, configs)
- Manage avr-asterisk repository
- Configure GitHub Actions CI/CD workflows
- Manage Docker Hub image publishing

### Docker/Infrastructure
- Run docker-compose commands for testing
- Build and push Docker images
- Validate Docker Compose YAML syntax
- Check Docker Hub for image versions and sizes

### File System
- Create and edit Docker Compose YAML files
- Manage Asterisk configuration files
- Edit .env.example files
- Manage GitHub Actions workflow YAML files
